Project created to check my own library

In this program I import my library and the readline library to give on terminal some inputs.

With two numbers that the user choose and the operation choosen we call the library function to do 
this mathematical operation.

Instructions to instal:

Install library: npm install calculator_ts_jose_sanz

Technologies: 

node, typescript

In this exercise I have uploaded the library to NPM with this commands:


npm login
npm pack
npm publish
